Here I want to share some of my work. Although I spend my time on different projects, I want to add some highlights of passion projects as well as professional ones.

Dashboards

  • Paraguayan Health Professionals Dashboard

    • Project Goal: Visualize open data regarding health professionals disponibility accross Paraguay.
    • Process: Extract, load, and transform open data from the Paraguayan Statistical National Institute.
    • Results: Tableau dashboard with easily accesable data and visualizations.

  • Bike Share User Data Analysis

    • Project Goal: To differentiate behavior between two user segments: casual riders versus annual subscribers.
    • Process: The data was provided by the company, the preparation and analysis was done with R.
    • Results: Dashboard describing user's behavior.

  • Paraguayan Health Professionals Dashboard

    • Project Goal: To make the complex 2024 Paraguayan national budget readily accessible and comprehensible to a wider audience.
    • Process: R programming, along with libraries like pdftools and tidyverse, was used to extract, clean, and structure the data.
    • Results: Users can gain valuable insights into where the Paraguayan government is allocating its resources.

  • Residential Tree Planting Program Dashboard

    • Project Goal: Process, visualize and track the development of the program.
    • Process: Create datasets, ArcGIS 123 collection tool, and dashboard.
    • Results: Up-to-date dashboard with relevant KPI for the project.

  • Custom Soil Analysis Report

    • Project Goal: Process and visualize soil analysis report.
    • Process: Extract, transform, and load data to geodatbase.
    • Results: Interactive web map with enhance visualizations using Leaflet JS and Chart JS.
