As a Geospatial Data Scientist I have processed data to create actionable outputs. I have worked with different tools like R, Python, and ArcGIS Pro. Throughout my career I have worked with different visualization tools. For non-geographic data, I worked with R Studio, Python and Tableau.

Proficient across the entire Data Pipeline, I excel in designing robust data schemas, building efficient data collection applications, expertly processing geospatial data, and developing impactful reports and dashboards.
